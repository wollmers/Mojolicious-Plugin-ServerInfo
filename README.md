# SYNOPSIS

    $app->plugin('Mojolicious::Plugin::ServerInfo');

# DESCRIPTION

[Mojolicious::Plugin::ServerInfo](https://metacpan.org/pod/Mojolicious::Plugin::ServerInfo) is a Mojolicious-Plugin.

It creates a route

    /serverinfo
    

where detailed info is dispalyed in formatted HTML.
